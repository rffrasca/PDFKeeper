Updating PDF Text Columns in the Database

PDFKeeper 7.0.0 introduced new table columns in the database to store Text Annotations and Text that is extracted from the PDF during the upload while PDFKeeper 7.1.0 introduced text extraction for “image-only” PDF documents using OCR (Windows 10 only). Prior to these additions, any existing documents in the database will be missing data in these new columns. With this procedure, existing documents in the database can be updated.


To update the PDF Text Annotations and Text in the Database for one or more retrieved documents, perform the following steps:

  1. In the documents list, click on the check box for each document that you wish to update. To select all documents in the list, select the File menu, then Select, and then All; or press Alt + F, then L, and then A on the keyboard.

  2. Select the Tools menu, and then select Update PDF Text Columns; or select the Update PDF Text Columns tool bar icon; or press Alt + T, and then U on the keyboard.

  3. When prompted, select Yes or press Enter on the keyboard to perform the update.

Additional Information:

  • You can navigate the documents list using the up and down arrows on the keyboard and check/uncheck a document using the space bar on the keyboard.

  • To uncheck all documents in the list, select the File menu, then Select, and then None; or press Alt + F, then L, and then N on the keyboard.

  • Depending on how many documents are selected, the update operation can be time consuming.
