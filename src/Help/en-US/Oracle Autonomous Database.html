Oracle Autonomous Database

This topic provides the steps that need to be performed for first time setup of Oracle Autonomous Database by the individual responsible for the administration of the database for use with PDFKeeper. It is assumed that an Autonomous Database has been created in the Oracle Cloud Infrastructure, you have familiarized yourself with the documentation specific to the Oracle Autonomous Database and version being used, and assume full responsibility of the database.

Note: only TLS authentication is supported by PDFKeeper. Mutual (mTLS) authentication support will be added in a later version.
