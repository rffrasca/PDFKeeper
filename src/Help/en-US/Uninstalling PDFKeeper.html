Uninstalling PDFKeeper

PDFKeeper can be removed from a computer using Installed Apps (Windows 11), Apps and Features (Windows 10), or Programs and Features (Windows 11/10). The uninstall process can be performed by a standard Windows user and does not require elevated privileges.

Items not removed during uninstall

The following folders (including their files) and registry keys remain on the system after uninstalling PDFKeeper:

  • %LOCALAPPDATA%\Robert_F._Frasca\PDFKeeper.exe_Url_<hash> (may contain multiple folders)

  • %APPDATA%\Robert F. Frasca\PDFKeeper

  • HKEY_CURRENT_USER\Software\Robert F. Frasca\PDFKeeper

  • HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Robert F. Frasca\PDFKeeper

  • Database installation and client software
