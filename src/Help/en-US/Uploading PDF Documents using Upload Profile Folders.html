Uploading PDF Documents using Upload Profile Folders

PDF documents can be uploaded using Upload Profile folders from outside of PDFKeeper using the following methods:

  • From Web Browser/application, save the PDF document into the desired sub-folder inside PDFKeeper Upload located on the Windows Desktop and in the Windows Downloads folder.

  • Using Windows Explorer, copy the PDF document(s) that you wish to upload and paste into the desired sub-folder inside PDFKeeper Upload located on the Windows Desktop and in the Windows Downloads folder. You can also copy one or more folders that contain only PDF documents.

Additional Information:

  • PDFKeeper must be open for the PDFKeeper Upload folder shortcut to be available on the Windows Desktop and in the Windows Downloads folder.

  • The selected sub-folder located in PDFKeeper Upload must have been created by PDFKeeper.

  • The Title, Author, Subject, Keywords, Category, and flag state will be applied to the PDF document(s) using the profile for the selected sub-folder.

  • USER password protected PDF documents are not supported by PDFKeeper. If you have the legal right to remove the password, you can find plenty of information on the Internet to assist you.

  • To automate the uploading of PDF documents, see Automating with the PDFKeeper Upload Process.
