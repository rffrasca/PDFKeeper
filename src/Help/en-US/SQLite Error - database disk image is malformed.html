When finding Documents by Text, you encounter the database disk image is malformed error. This error can occur when performing a find sometime after one or more documents were deleted from the SQLite database. If one or more documents were deleted sometime before performing the find, then the fix in this topic should be applied to rebuild the full-text search index and recreate the synchronization triggers.

Before applying the fix, please make sure that no transactions are running against the database. Next, it is highly recommended to backup the database file PDFKeeper.sqlite. By default, the database file exists in %APPDATA%\Robert F. Frasca\PDFKeeper unless it was moved to an alternate location.
