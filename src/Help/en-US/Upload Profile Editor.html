Upload Profile Editor

The Upload Profile Editor dialog is where the properties are defined for the upload profile being added or edited.

  1. Enter or edit the Name of the profile. Skip if you wish to set the Name to the combined values of Author and Subject.

  2. Enter or edit the Title, or select a predefined token <Date>, <Date_Time>, <FileName>, or <Title>. Note: When using <Title> to use the existing Title from the PDF, each PDF must include a Title in its document properties; otherwise, the PDF will be rejected by the upload process.

  3. To use the Author already stored in the PDF, select Use existing Author from PDF. Otherwise, you may enter, edit, or select an Author manually. The Author combo box is linked to the database, allowing you to choose from existing Authors. Note: When using the existing Author from the PDF, each PDF must include an Author in its document properties; otherwise, the PDF will be rejected by the upload process.

  4. To use the Subject already stored in the PDF, select Use existing Subject from PDF. Otherwise, you may enter, edit, or select a Subject manually. The Subject combo box is linked to the database, allowing you to choose from existing Subjects. Note: When using the existing Subject from the PDF, each PDF must include a Subject in its document properties; otherwise, the PDF will be rejected by the upload process.

  5. To set the Name of the profile to the combined values of Author and Subject, select Set Name to Author and Subject.

  6. To set the Name of the profile to the Subject, select Set Name to Subject.

  7. To use the Keywords already stored in the PDF, select Use existing Keywords from PDF. Otherwise, you may enter or edit the Keywords (optional) manually.

  8. Enter, edit, or select the Category (optional). The Category combo box is linked to the database to allow for selecting an existing Category.

  9. Select the Tax Year (optional).

  10. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to list bills that require payment.

  11. Check or uncheck OCR PDF pages containing both text and image data. When checked, each PDF page that contains one or more images will be processed by OCR to extract text. This is useful when the PDF contains pages with embedded images that contain text that would normally not be processed.

  12. To save the profile, select OK or press Enter on the keyboard. To discard the changes, select Cancel or press Esc on the keyboard.
