Document Notes

The Notes text box on the main form is where information related to the selected document is entered. In addition, each entry can be stamped with the Date and Time, and user account name. All Document Notes are indexed when saved, making the text searchable.

Editing Notes:

  1. Select the document in the Documents list.

  2. Select the Notes tab, and then select the text box inside the Notes tab page.

  3. To append a stamp with the Date and Time, and user account name; either select the Edit menu, and then select Append Date/Time; or select the Append Date/Time tool bar icon; or press Ctrl + Alt + D on the keyboard.

  4. To append text from a text file, either select the Edit menu, and then select Append Text; or select the Append Text tool bar icon; or press Ctrl + Alt + T on the keyboard

  5. After making the necessary edits, save the changes by either selecting the File menu, and then select Save; or select the Save tool bar icon; or press Ctrl + S on the keyboard.

Saving Notes to a text file:

  1. Select the document in the Documents list.

  2. Select the Notes tab, and then select the text box inside the Notes tab page.

  3. Select the File menu, and then select Save As; or press Alt + F, and then E on the keyboard.

  4. Select the folder location and enter the filename to save as.

  5. Select Save or press Enter on the keyboard.

Printing Notes:

  1. Select the document in the Documents list.

  2. Select the Notes tab, and then select the text box inside the Notes tab page.

  3. Select the File menu, and then select Print; or select the Print tool bar icon; or press Ctrl + P on the keyboard. As an option, to preview before printing, select the File menu, and then select Print Preview; or press Alt + F, and then V on the keyboard.

Additional Information:

  • When editing Notes, the Documents list is disabled.

  • If you need to restore the text box contents back to the state when the document was selected, basically undoing all of the edits made, select the Edit menu, and then select Restore; or select the Restore tool bar icon; or press Ctrl + R on the keyboard.

  • In a multi-user database environment, the editing and saving of Document Notes will be disabled if your database user account has not been granted update access to the docs table.
