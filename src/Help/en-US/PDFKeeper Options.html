PDFKeeper Options

The following options can be changed from the Options dialog:

Select last row when listing documents.

  • When checked, the last row in the documents list will be selected following a find.

  • When unchecked, the first row in the documents list will be selected following a find.

Show PDF documents with default application.

  • When checked, any PDF associated with a document in the list will be opened using the bundled, third-party viewer.

  • When unchecked, any PDF associated with a document in the list will be opened using the user's default PDF application.

Find flagged documents on startup.

  • When checked, Flagged Documents will be shown in the documents list on the startup of PDFKeeper.

  • This option, when checked will uncheck Show all documents on startup.

Show all documents on startup.

  • When checked, All Documents will be shown in the documents list on the startup of PDFKeeper.

  • This option is only available when using the SQLite local database platform.

  • This option, when checked will uncheck Find flagged documents on startup.

Compact database after deleting selected documents.

  • When checked, the database will compacted after deleting selected documents.

  • This option is only available when using the SQLite local database platform.

Setting Documents View Column Header Aliases

PDFKeeper allows you to customize the names used for several column headers in the Documents View. These custom names—called aliases—let you tailor the terminology to match your organization’s workflow or personal preferences. Aliases apply only to the Documents View and do not affect terminology used elsewhere in the application.

You can set aliases for the following fields:

  • Author

  • Subject

  • Category

  • Tax Year

To reset a field to its default value, simply clear the custom text. To restore all fields to their default values at once, select Reset Aliases.

Aliases are stored in a file named aliases.json that is located in %APPDATA%\Robert F. Frasca\PDFKeeper. This file is portable and can be transferred to any number of computers. When copying aliases.json into the same folder on the target computer, be sure that PDFKeeper is closed on the target system before replacing the file.
