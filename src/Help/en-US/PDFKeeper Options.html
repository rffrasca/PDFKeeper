PDFKeeper Options

The following options can be changed from the Options dialog:

Select last row when listing documents.

  • When checked, the last row in the documents list will be selected following a find.

  • When unchecked, the first row in the documents list will be selected following a find.

Show PDF documents with default application.

  • When checked, any PDF associated with a document in the list will be opened using the bundled, third-party viewer.

  • When unchecked, any PDF associated with a document in the list will be opened using the user's default PDF application.

Find flagged documents on startup.

  • When checked, Flagged Documents will be shown in the documents list on the startup of PDFKeeper.
