Upload Profiles

Upload profiles can be added, edited, and deleted; each having its own properties. These properties will be applied to each PDF during the upload process.

  • From the PDFKeeper main form, select the Tools menu, and then select Upload Profiles; or select the Upload Profiles tool bar icon; or press Alt + T and then C on the keyboard. Once the Upload Profiles dialog opens, you can Add, Edit, and Delete Upload Profiles.
