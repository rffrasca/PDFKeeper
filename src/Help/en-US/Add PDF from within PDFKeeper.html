Add PDF from within PDFKeeper

On the Add PDF dialog, perform the following steps:

  1. To view the selected PDF with the bundled viewer, select View or press Alt + V on the keyboard.

  2. Enter the PDF Title. To set the Title to the file name of the selected PDF, select Set Title to PDF File Name.

  3. Enter or select the Author of the PDF. The Author combo box is linked to the database to allow for selecting an existing Author.

  4. Enter or select the Subject of the PDF. The Subject combo box is linked to the database to allow for selecting an existing Subject.

  5. If necessary, enter PDF Keywords.

  6. If necessary, enter or select the Category of the PDF. The Category combo box is linked to the database to allow for selecting an existing Category.

  7. If necessary, select the Tax Year of the PDF.

  8. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to list bills that require payment.

  9. Check or uncheck OCR PDF pages containing both text and image data. When checked, each PDF page that contains one or more images will be processed by OCR to extract text. This is useful when the PDF contains pages with embedded images that contain text that would normally not be processed.

  10. Check or uncheck Delete selected PDF when added. When checked, the selected PDF is deleted to the Windows Recycle Bin.

  11. Select OK or press Enter on the keyboard to write the modified version of the PDF that includes the specified Information Properties and external metadata XML to %APPDATA%\Robert F. Frasca\PDFKeeper\UploadStaging. If the selected PDF contained an OWNER password, the password will not be written to the modified PDF. To cancel, select Cancel or press the Esc key on your keyboard.
