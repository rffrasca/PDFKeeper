Add PDF from within PDFKeeper

On the Add PDF dialog, perform the following steps:

  1. To view the selected PDF with the bundled viewer, select View or press Alt + V on the keyboard.

  2. Enter the PDF Title. To set the Title to the file name of the selected PDF, select Set Title to File Name or press Alt + F on the keyboard.

  3. Enter or select the Author of the PDF. The Author combo box is linked to the database to allow for selecting an existing Author.

  4. Enter or select the Subject of the PDF. The Subject combo box is linked to the database to allow for selecting an existing Subject.

  5. If necessary, enter PDF Keywords.

  6. If necessary, enter or select the Category of the PDF. The Category combo box is linked to the database to allow for selecting an existing Category.

  7. If necessary, select the Tax Year of the PDF.

  8. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to list bills that require payment.

  9. Check or uncheck Delete selected PDF when added. When checked, the selected PDF is deleted to the Windows Recycle Bin.

  10. Check or uncheck OCR each PDF page containing text and image data. When checked, each PDF page that contains one or more images will be processed by OCR to extract text. This is useful when the PDF contains pages with embedded images that contain text that would normally not be processed.

  11. Select Save or press Alt + S on the keyboard to write the modified version of the PDF that includes the specified Information Properties and extension properties XML to %APPDATA%\Robert F. Frasca\PDFKeeper\UploadStaging. If the selected PDF contained an OWNER password, the password will not be written to the modified PDF. Selecting Save will need to be repeated if changes are made following a Save and prior to selecting Add.

  12. If necessary, you can view the modified version of the selected PDF with the bundled viewer either by selecting Preview or by pressing Alt + P on the keyboard.

  13. If no further changes need to be made and you wish to upload the file, select Add or press Alt + A on the keyboard. To cancel and discard the modified PDF, select Cancel or press the Esc key on your keyboard.
