PDF Attachments Extraction

To extract all attachments in the PDF associated with a selected document into a folder or ZIP file, perform the following steps:

  1. Select the document in the Documents list, then select the File menu, and then select Extract All Attachments; or press Ctrl + E on the keyboard; or select the Extract All Attachments tool bar icon.

  2. When prompted, select Yes to extract to a ZIP file or No to extract to a folder.

  3. Select the folder location and enter the ZIP filename to extract into, and then select Save or press Enter on the keyboard to perform the extraction; or select the folder location where to extract into, and then select OK or press Enter on the keyboard to perform the extraction.
