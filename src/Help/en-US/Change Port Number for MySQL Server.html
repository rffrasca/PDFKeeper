Change Port Number for MySQL Server

This topic provides the steps for changing the port number used by PDFKeeper when connecting to MySQL server. By default, PDFKeeper uses port 3306, the MySQL Server default.

If you need to change the port number, perform the following steps:

  1. From Windows, right click on the Start button and then select Run.

  2. Paste %LOCALAPPDATA%\Programs\PDFKeeper\ConnectionSettingManager.cmd into the combo box and then select OK.

  3. Select option 3.

  4. When prompted, enter or paste the new port number.
