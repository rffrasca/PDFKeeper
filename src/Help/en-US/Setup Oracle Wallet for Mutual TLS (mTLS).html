Setup Oracle Wallet for Mutual TLS (mTLS)

This topic provides steps for setting up an Oracle Wallet on the client for connecting to the Oracle Autonomous Database using Mutual TLS (mTLS) authentication. It is assumed that the individual responsible for the administration of the database has downloaded and securely provided the Oracle Wallet zip file using these instructions.

To setup the Oracle Wallet on the client, perform the following steps:

  1. Extract the provided Wallet_databasename.zip into an empty folder. After extracting the zip file, make sure to secure it to prevent unauthorized database access.

  2. To secure the wallet folder from unauthorized access, set the permissions on the wallet folder to allow the file owner to have Read and Write permissions while all other users have no permissions.
