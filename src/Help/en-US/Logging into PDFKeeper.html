Logging into PDFKeeper

These steps are only required when using PDFKeeper with a compatible, multi-user (on-prem or cloud) database management system.

On the PDFKeeper Login form:

  1. Enter your database User name.

  2. Enter your database Password.

  3. Enter the Data source specified as host:port/service_name for an on-prem database or the TNS name or connection string from one of the available Connection Strings for the Autonomous Database.

  4. When the Data source contains oraclecloud.com, the Select Oracle Wallet link will be available to select an Oracle Wallet for connecting to the Oracle Cloud Database with Mutual TLS (mTLS) authentication. To setup Mutual TLS (mTLS) authentication, select the link to choose the folder location where the Oracle Wallet is located, and then select OK or press Enter on the keyboard.

  5. Select OK or press Enter on the keyboard.

Additional Information for a client-server database connection:

  • host is the host name or IP address of the database server computer. If connecting to a database located on the same computer, the host name or IP address would be localhost or 127.0.0.1.

  • port is the listening port number on the database server. If not specified, the default port number 1521 is assumed.

  • service_name is the service name of the database instance or pluggable database to access. For Oracle Database Express Edition, if not specified, the default service name XE is assumed.
