Getting Started

PDFKeeper requires a database for document storage. It can be implemented for use in a single user or compatible, multi-user (on-prem or cloud) database environment. If implementing for use in a multi-user (on-prem) environment, it is recommended that the database be installed on a dedicated system that is capable of handling the expected load.
