Set Tax Year on Documents

To set the tax year on one or more retrieved documents, perform the following steps:

  1. In the Documents list, click on the check box for each document that you wish to set the tax year on. To select all documents, select the Documents menu, then Select, and then All; or press Alt + D, then S, and then A on the keyboard.

  2. Select the Documents menu, and then select Set Tax Year; or press Ctrl + Shift + T on the keyboard.

  3. In the Set Tax Year dialog, select the desired tax year or blank item to clear, and then select OK or press Enter on the keyboard to perform the update.

Additional Information:

  • You can navigate the Documents list using the up and down arrows on the keyboard and check/uncheck a document using the space bar on the keyboard.

  • To uncheck all documents, select the Documents menu, then Select, and then None; or press Alt + D, then S, and then N on the keyboard.
