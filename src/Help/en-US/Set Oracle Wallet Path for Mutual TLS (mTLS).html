Set Oracle Wallet Path for Mutual TLS (mTLS)

This topic provides steps for setting the path to the Oracle Wallet that was downloaded onto the client for connecting to the Oracle Autonomous Database using Mutual TLS (mTLS) authentication with PDFKeeper. It is assumed that the individual responsible for the administration of the database has downloaded and securely provided the Oracle Wallet zip file using these instructions and that the zip file has been extracted to a secure folder following these instructions.

To set the Oracle Wallet path on the client, perform the following steps:

  1. From Windows, right click on the Start button and then select Run.

  2. Paste %LOCALAPPDATA%\Programs\PDFKeeper\ConnectionSettingManager.cmd into the combo box and then select OK.

  3. Select option 2.

  4. When prompted, enter or paste the Oracle Wallet folder absolute path name.
