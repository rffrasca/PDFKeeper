Change Document Flag State

The Flag Document menu item on the main form is used to change the flag state for the selected document. When the Flag Document menu item is checked, the selected document will be flagged.

Change Document Flag state:

  1. Select the document in the Documents list.

  2. Select the Edit menu, and then select Flag Document; or press Alt + E, and then F on the keyboard.

Additional Information:

  • In a multi-user database environment, the controls referenced in step 2 will be disabled if your database user account has not been granted update access to the docs table.
