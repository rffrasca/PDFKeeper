Oracle Database (On-Prem)

This topic provides the steps that need to be performed for first time setup of Oracle Database by the individual responsible for the administration of the database for use with PDFKeeper. It is assumed that Oracle Database has been installed on the computer that will host the database, you have familiarized yourself with the documentation specific to the Oracle Database edition and version being used, and assume full responsibility of administrating the database.
