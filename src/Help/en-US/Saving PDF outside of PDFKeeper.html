Saving PDF outside of PDFKeeper

To save the PDF for the selected document to a folder location outside of PDFKeeper, perform the following steps:

  1. Select the document in the Documents list.

  2. Select the File menu, and then select Save As; or press Alt + F, and then E on the keyboard.

  3. Select the folder location and enter the filename to save as.

  4. Select Save or press Enter on the keyboard.
