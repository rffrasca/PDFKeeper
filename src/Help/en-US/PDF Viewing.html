PDF Viewing

To view the PDF associated with a selected document, perform any of the following methods:

  • Double click on the document in the Documents list.

  • Select the document in the Documents list, and then select the File menu, and then select Open.

  • Select the document in the Documents list, and then press Ctrl + O on the keyboard.

  • Select the document in the Documents list, and then select the Open tool bar icon.

To open the PDF for each selected (checked) document up to a maximum of 12, perform the following steps:

  1. In the Documents list, click on the check box for each document that you wish to open. To select all documents, select the File menu, then Select, and then All; or press Alt + F, then L, and then A on the keyboard. Keep in mind, only the PDF documents for the first 12 selected (checked) documents will be opened.

  2. Select the File menu, and then select Open; or press Ctrl + O on the keyboard; or select the Open tool bar icon.

Additional Information:

  • By default, the PDF is opened with the bundled, third-party viewer. To open with the user's default PDF application, see this topic.

  • You can navigate the Documents list using the up and down arrows on the keyboard and check/uncheck a document using the space bar on the keyboard.

  • To uncheck all documents, select the File menu, then Select, and then None; or press Alt + F, then L, and then N on the keyboard.
