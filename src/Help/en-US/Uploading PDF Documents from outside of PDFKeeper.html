Uploading PDF Documents from outside of PDFKeeper

PDF documents can be uploaded from outside of PDFKeeper using the following methods:

  • From Web Browser/application, save the PDF into PDFKeeper Upload, located on the Windows Desktop and in the Downloads folder.

  • Using Windows Explorer, copy or move the PDF document(s) that you wish to upload and paste into PDFKeeper Upload, located on the Windows Desktop and in the Downloads folder. You can also copy one or more folders that contain only PDF documents.

Additional Information:

  • PDFKeeper must be open for the PDFKeeper Upload folder shortcut to be available on the Windows Desktop and in the Downloads folder.

  • PDF document(s) must meet the criteria specified in the Uploading PDF Documents topic; otherwise, any that don't will be rejected by the Upload process.

  • In a multi-user database environment, the Upload process will reject all PDF document(s) added into PDFKeeper Upload if your database user account has not been granted insert access to the docs table.

  • To automate the uploading of PDF documents, see Automating with the PDFKeeper Upload Process.
