Setup Multi-User Database

PDFKeeper can be implemented for use in a multi-user (on-premises or cloud) database environment. If implementing for use in a multi-user (on-premises) environment, it is recommended that the database be installed on a dedicated system that is capable of handling the expected load.

If installing SQL Server or SQL Server Express, please follow the instructions in this topic before setting up the database for PDFKeeper.
