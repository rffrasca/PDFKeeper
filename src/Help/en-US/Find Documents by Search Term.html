Find Documents by Search Term

Documents can be retrieved by Search Term, meaning any plain text that was extracted from each PDF when uploaded to the database, including metadata, text annotations, and any notes added to the document after being uploaded. When the PDF being uploaded is “image-only” and the operating system is Windows 10, the text will be extracted from the PDF using OCR.
