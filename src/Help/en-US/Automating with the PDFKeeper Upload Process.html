Automating with the PDFKeeper Upload Process

Automated uploading of PDF documents created by PDF printer applications, Scanners, and batch processes can be achieved with PDFKeeper using the following method:

  • Configure the application/device/process to save PDF documents into %APPDATA%\Robert F. Frasca\PDFKeeper\Upload or an upload profile folder. To save directly to %APPDATA%\Robert F. Frasca\PDFKeeper\Upload all PDF documents must contain a Title, Author, and Subject.

Additional Information:

  • PDFKeeper must be open for the upload process to execute. If PDFKeeper is not running, files will accumulate until the next time PDFKeeper is opened.

  • If the PDF documents that need to be uploaded are created on another system, create a Scheduled Task in Windows to call a script to copy the PDF documents into %APPDATA%\Robert F. Frasca\PDFKeeper\Upload or an upload profile folder.

  • When using an upload profile folder, the Title, Author, Subject, Keywords, Category, Tax Year, and flag document state will be applied to the PDF document(s) using the properties defined for that folder.
