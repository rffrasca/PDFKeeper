Document Keywords

The Keywords (read only) text box on the main form is where the Keywords from the selected document. These keywords are read from the PDF when uploaded to the database.

Viewing Keywords:

  1. Select the document in the Documents list.

  2. Select the Keywords tab.

Saving Keywords to a text file:

  1. Select the document in the Documents list.

  2. Select the Keywords tab, and then select the text box inside the Keywords tab page.

  3. Select the File menu, and then select Save As; or press Alt + F, and then E on the keyboard.

  4. Select the folder location and enter the filename to save as.

  5. Select Save or press Enter on the keyboard.
