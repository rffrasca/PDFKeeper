MySQL

This topic provides the steps that need to be performed for first time setup of MySQL by the individual responsible for the administration of the database for use with PDFKeeper. It is assumed that MySQL has been installed on the computer that will host the database, you have familiarized yourself with the documentation specific to the MySQL version being used, and assume full responsibility of administrating the database.

At this time, PDFKeeper has only been tested with MySQL Community Server. This edition is freely downloadable and supported by an active community.
