Installing Oracle ODP.NET

This topic provides the installation steps for Oracle ODP.NET that is required for PDFKeeper to connect to Oracle Database.

This version of PDFKeeper requires Oracle ODP.NET 23.26.100 (4.122.23.1) to be installed. Older versions of ODP.NET, Managed Driver and Oracle Data Access Components used by previous versions of PDFKeeper should be uninstalled if no longer needed by other applications.

  1. Go to Oracle.ManagedDataAccess 23.26.100 on NuGet.

  2. Select License Info and review the license file.

  3. Select Open in NuGet Package Explorer.

  4. Under Contents expand lib, and then expand net472.

  5. Double click Oracle.ManagedDataAccess.dll.

  6. Save Oracle.ManagedDataAccess.dll to the Downloads folder on your computer.

  7. When prompted, select Save.

  8. Open the Downloads folder, and then open the file Properties of Oracle.ManagedDataAccess.dll.

  9. Select Unblock, and then Apply.

  10. Verify the Digital Signature of the file. The Name of signer should be Oracle America, Inc.

  11. Select Oracle America, Inc., and then Details. Verify the digital signature is OK.

  12. Select OK to close Digital Signature Details.

  13. Select OK to close Properties.

  14. Move Oracle.ManagedDataAccess.dll to the folder of your choice.

  15. From Windows, right click on the Start button and then select Run.

  16. Paste %LOCALAPPDATA%\Programs\PDFKeeper\ConnectionSettingManager.cmd into the combo box and then select OK.

  17. Select option 1.

  18. When prompted, enter or paste the Oracle ODP.NET folder absolute path name.
